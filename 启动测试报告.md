## 📊 修复的问题

### 问题 1：初始化脚本引用已删除的模块

**问题描述：**
`backend/app/scripts/initialize.py` 中引用了已删除的模块：
- `ParamsModel`（参数配置模块）
- `DictTypeModel`, `DictDataModel`（字典管理模块）

**修复方案：**
1. 删除了对已删除模块的 import 语句
2. 从 `prepare_init_models` 列表中移除了这些模型
3. 删除了字典类型表的特殊处理逻辑

**修改文件：**
- `backend/app/scripts/initialize.py`

**修改内容：**
```python
# 删除的 import
- from app.api.v1.module_system.params.model import ParamsModel
- from app.api.v1.module_system.dict.model import DictTypeModel, DictDataModel

# 修改后的初始化模型列表
self.prepare_init_models = [
    MenuModel,
    DeptModel,
    RoleModel,
    PositionModel,
    UserModel,
    UserRolesModel,
]

# 删除的字典处理逻辑
- elif table_name == "sys_dict_type": ...
- elif table_name == "sys_dict_data": ...
```
