# Admin Web

Vue 3 + Vite + TypeScript frontend for the admin system.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Structure

```txt
src/
  api/          request modules
  assets/       static assets
  components/   shared components
  layouts/      app layouts
  router/       route setup
  store/        state modules
  styles/       global styles
  views/        pages
```
